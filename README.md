# Student Performance ML

A beginner machine learning project using the student performance dataset to demonstrate both regression and classification.

## Models Included

### Regression
Predicts math scores using reading and writing scores.

### Classification
Predicts whether a student passes or fails in math.

## Technologies Used
- Python
- pandas
- scikit-learn
- matplotlib

## Concepts Used
- Linear Regression
- Logistic Regression
- Train-test split
- Mean Absolute Error
- Accuracy
- Confusion Matrix

## Purpose
Built to learn core machine learning concepts and model evaluation.
